Um curso que explora os benefícios de utilizar o Git como ferramenta de controle de versão para projetos em quaquer linguagem, em qualquer plataforma.

  • Permite a edição concorrente de arquivos do projeto
  • Não depende de uma conexão ativa com um servidor.
